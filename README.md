# Calculadora Infantil

Nesse repositório está depositado o código da Calculadora Infantil.
Esta Calculadora utiliza o console do computador para rodar, ela tem como objetivo interagir de maneira didatica e interativa com crianças com o intuito de promover um melhor aprendizado da Matemática.
